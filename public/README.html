Design System
3D Perspective System
Every section uses perspective: 1400px with preserve-3d cards that tilt on mouse movement via spring physics. Cards have directional bevel lighting simulating a top-left light source.
Chiaroscuro Contrast
Alternating section-lit / section-dark bands create bright content islands against a deep void background. A fixed vignette layer darkens screen edges to focus the eye on center content.
Physics-Based Cursor
Blob cursor stretches in the direction of movement. Samples real-time luminance under the cursor — glowing neon battery effect in dark areas, directional drop shadow in light areas.
Animated Theme Toggle
Sliding pill switch with moon + twinkling stars (dark mode) or sun + radiating rays (light mode). Full light mode palette with proper WCAG contrast ratios.
